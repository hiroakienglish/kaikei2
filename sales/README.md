# 売上管理帳 (Sales App)

シンプルな売上管理Webアプリです。  
商品別・日別の売上を記録・分析できます。

## 🔧 機能

- 売上入力（商品・数量・単価）
- 履歴検索
- 売上分析（グラフ）
- CSVインポート / エクスポート
- ローカル保存（localStorage）

## 🚀 デモ

GitHub Pagesで公開可能

## 🛠 使用技術

- HTML / CSS / JavaScript
- Chart.js

## 📦 使い方

1. index.html をブラウザで開くだけ

## 🌐 公開方法（GitHub Pages）

1. GitHubにリポジトリ作成
2. index.html をアップロード
3. Settings → Pages → Branch: main → /root
4. URLが発行される

## ⚠ 注意

- データはブラウザに保存されます（localStorage）
- 別の端末では共有されません
